## 🎯 Visão Geral

O **IncentivaBR** é uma plataforma exclusiva e legalmente protegida (Registro INPI nº BR512025000647-0) projetada especificamente para **servidores e empregados públicos** de todas as esferas (federal, estadual, municipal), incluindo militares, aposentados e pensionistas.

### 🚀 Proposta de Valor

Facilitar o exercício do direito de destinar parte do Imposto de Renda (IR) devido para projetos e causas sociais, utilizando os incentivos fiscais previstos em lei, com **total segurança jurídica e fiscal** através do conceito inovador de **"Sistema de Compliance Fiscal Ativo com Garantia de Conformidade"**.

## 🚀 Como Executar

### Pré-requisitos
- Python 3.11+
- Git

### Instalação Rápida

1. **Clone o repositório**
```bash
git clone https://github.com/casdfteste/incentivabr-mvp-premium.git
cd incentivabr-mvp-premium
```

2. **Execute com Docker (Recomendado)**
```bash
docker-compose up -d
```

3. **Ou execute manualmente**
```bash
# Backend services (em terminais separados)
cd backend/otimizacao_fiscal_service && python src/main.py
cd backend/plataforma_projetos_service && python src/main.py
cd backend/gestao_fundos_service && python src/main.py
cd backend/compliance_service && python src/main.py
cd backend/painel_controle_service && python src/main.py

# Frontend
cd frontend && python -m http.server 8080
```

4. **Acesse a aplicação**
```
http://localhost:8080
```
